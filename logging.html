Logging Training

Dataclasses for structuring data from test runs of an agent in an environment.

class src.logging.Episode(steps: list[~src.logging.Step] = <factory>, parameters: dict[str, float] = <factory>, outcome: str = '')

Bases: object

An episode is a single run of an agent in an environment. A collection of time steps.

steps

Sequence of (time) Steps.

Type:

list[Step]

parameters

The parameters of the environment when the episode was run. e.g. {“mass”: 0.1, “length”: 0.5}. The keys are the names of the parameters. This needs to be stored as the parameters vary between episodes. They are sampled from the distributions defined by the user.

Type:

dict[str, float]

outcome

The outcome of the episode. “Success” or “Failure”.

Type:

str

class src.logging.Step(time_step: int, obs: Tensor, reward: float, action: int, render: ndarray)

Bases: object

A single time step of an episode. A collection of data from a single step of an agent in an environment.

time_step

The time step of the environment when the step was taken. (0-indexed). Ordinal, not in seconds.

Type:

int

obs

The observation of the environment when the step was taken. Depends on the environment.

Type:

Tensor

reward

The reward given to the agent from the environment when the step was taken.

Type:

float

action

The action taken by the agent when the step was taken. Corresponds to the environment’s action space.

Type:

int

render

The rendered image of the environment when the step was taken.

Type:

ndarray
