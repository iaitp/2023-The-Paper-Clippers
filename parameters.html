class src.parameter.Parameter(name: str, default: float, min: float, max: float, unit: str, lib_ref: str | None = None)

A dataclass for structuring the user adjustable parameters of an environment.

float

Type:

float

The maximum value of the parameter. e.g. 1.0.

Type:

float

unit

The unit of the parameter. e.g. “kg”.

Type:

str

lib_ref

Sometimes the name of the parameter in the environment is different to the name used in the streamlit app. This optional parameter acts as a lookup. It means the values of parameters in a specific Episode can be recovered. E.g. for parameter in parameters:

value = env.get_attr(parameter.lib_ref)

Type:

str

default: float
get_lib_ref() → str

A getter function for the lib_ref attribute. If the environment and the app use the same name for the parameter, then the lib_ref attribute is not specified and this returns the key.

Returns:

The lib_ref attribute or the key. E.g. “masspole” or “mass”.
